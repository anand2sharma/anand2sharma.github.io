Asking Questions?

On initial observation and exploration it was time to think about what questions we can ask and get answersed by available data? I thought We can address these ones:

  • Which country has most consistent growth pattern?

Data Cleansing & Transformations

After forming basic questionaire, We have to transform raw data so we can compute and visualize it.

We will be using R to explore, understand and transform data. For those who are not familier with R, It is an open source simple programming launguge to manipulate data with tons of liberary availble for free. It has simple syntax so it should be readable even if you are not familier with this language.

We will use two R packages, ggplot2 for creating graphs and reshape for data manipulation and R Studio for development.

So lets get started:

First thing we need is to get the data needed for our exploration. Go to Gapminder's website and search for "Economic growth over the past 10 years" then download the excel file.
